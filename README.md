## [简介](https://github.com/mcuyun/RK3368/wiki)

2015CES发布，采用ARM Cortex-A53 八核心，主频最高1.5GHz，GPU为PowerVR G6110，Antutu跑分达35000



主要特点：双路SDIO 3.0

功能特点：HDMI 2.0、8M ISP

主要短板：USB

封装：BGA453( 19X19, 0.8mm pitch)
